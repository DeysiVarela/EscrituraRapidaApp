proyecto_interes: 
- El campo proyecto_interes puede tomar como valor un ID en formato GUID (ejemplo: xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx) o el texto "N/A".
- Usa la función consultar_proyectos_crm para obtener los ID válidos de los proyectos disponibles.
- El nombre del proyecto de interés proporcionado por el cliente no requiere una coincidencia exacta con los nombres de los proyectos disponibles. Por ejemplo, si el cliente menciona “lepar”, interpreta que se refiere al proyecto LEPARC 29. Si no se encuentra ninguna coincidencia relevante o el proyecto no está en la lista de proyectos disponibles, asigna el valor "N/A" como ID del proyecto.
- Cada vez que el cliente pregunte por un proyecto específico o solicite información sobre alguno, actualiza el campo proyecto_interes ejecutando la función de actualización con el ID correspondiente
